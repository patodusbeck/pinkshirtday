# Pink Shirt Day Quiz

Quiz interativo com 15 perguntas sobre respeito,
empatia e combate ao bullying.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Font Awesome

## Como usar

1. Baixe os arquivos
2. Abra `index.html`
3. O site funcionará automaticamente

## Personalização

### Alterar perguntas

Abra:

script.js

E edite o array:

const questions = []

### Alterar cores

Abra:

style.css

Modifique os tons de rosa:

#ff4f93
#ff86b6
#ffd6e8

### Adicionar animações

Edite:

@keyframes

No arquivo CSS.
